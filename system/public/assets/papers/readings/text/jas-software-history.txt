Jeffrey Alan Scudder — Software History (Pre-Aesthetic Computer)

Compiled from: CV, fact-check report, bin-sc.jas.life legacy archive, and Dropbox archive.

=== MAJOR SOFTWARE WORKS ===

1. thePRBAT — Polygon Replicating Bitmap Authoring Tool (2011)
   Type: Desktop application (Flash/SWF-based)
   Description: Early digital imaging software created during BFA at Ringling College. A bitmap authoring tool using polygon replication as its core interaction model.
   Archive: bin-sc.jas.life/Projects/thePRBAT/, Dropbox /thePRBAT/thePRBAT.swf

2. No Paint (2014–2020)
   Type: Web app (Rails, 2014) → WebGL prototype → iOS app (Cordova/Xcode, 2019–2020) → Vercel site
   Description: Constraint-based drawing application. Originally a Ruby on Rails web app at nopaint.org (2014), had a WebGL prototype stage, rebuilt as an iOS app (Cordova + Xcode) released in 2019–2020. The constraint: you can only paint, never erase. Full iOS project source preserved in Dropbox.
   Archive: nopaint-org.jas.life (original Rails app), Dropbox /Whistlegraph/No Paint/ (iOS source, nopaint.art deploy, stamps, playlists)
   Live: nopaint.art

3. JeffreyPaint (2014)
   Type: macOS desktop application (.dmg) + Unity version
   Description: A native macOS painting application with multiple builds. Also had a Unity port (JeffreyPaint-Unity).
   Archive: bin-sc.jas.life/Projects/JeffreyPaint/, Dropbox Source Code/JeffreyPaint-Unity-master.zip

4. Shrub (2014, revived 2017)
   Type: iOS application → 2017 web prototype
   Description: iOS app created while working at Linked by Air design studio in NYC (2013–2014). Had a 2017 web prototype version in Software Prototypes experiments.
   Archive: Dropbox /Archives/Project Archive/Shrub/, /Software Prototypes & Experiments 2017/shrub/

5. Finger Quilt (2016)
   Type: iOS application
   Description: Quilting-inspired touch drawing tool for iOS.
   Archive: Dropbox /Archives/Project Archive/Finger Quilt/

6. Traveller (2015)
   Type: Processing application → macOS .app builds
   Description: A generative/interactive Processing sketch with multiple standalone macOS builds (Traveller.app, Traveller 2.app, Traveller 3.app). Had "full-mode, big-dot" and "half-mode" variants. Also related to traveller.xn--q9jyb4c domain. A significant standalone artwork/tool.
   Archive: Dropbox /Archives/Project Archive/Traveller/ (source: Traveller.pde, 3 .app builds, PDFs, video recording)

7. Marker (2015)
   Type: Drawing tool
   Description: Drawing/marking software, source code preserved as "Marker (2015 edit)".
   Archive: Dropbox Source Code/Marker (2015 edit)/

8. Cricket
   Type: Creative tool
   Description: Software project with tutorial video. Source code and video archived.
   Archive: Dropbox Source Code/Cricket/, video: Cricket Tutorial (720p).mp4

9. ShelbySoft (date unknown, pre-2016)
   Type: macOS applications
   Description: A software suite including VoiceRecorder.app and VoiceWriter.app.
   Archive: bin-sc.jas.life/Projects/ShelbySoft/

10. TDA — Teacher Directed Activity (date unknown, pre-2016)
