=== AESTHETIC COMPUTER EVOLUTION ===

The AC platform evolved through 4 GitHub repos:

1. whistlegraph/system-ac (2021-08-10) — "My virtual computer system for loading and running disks."
   First commits: metronome, square wave instruments, basic 3D, animated notation.
   Started on Glitch.

2. whistlegraph/disks-ac (2021-08-10) — "A bunch of disks that are compatible with the system repo."
   Original disks: prompt.js, doodle.js, plot.js, pull.js, stage.js, starfield.js, tracker.js,
   whistlegraph.js, metronome-test.js

3. whistlegraph/2022.aesthetic.computer (2021-12-23) — "Compute aesthetically."
   Merged system+disks into one monorepo.

4. whistlegraph/aesthetic-computer (2022-12-23) — Current repo.
   "Copy all previous sources into the new repo" — third and final monorepo consolidation.
