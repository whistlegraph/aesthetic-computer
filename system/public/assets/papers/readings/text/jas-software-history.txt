Jeffrey Alan Scudder — Software History (Pre-Aesthetic Computer)

Compiled from: CV, fact-check report, and bin-sc.jas.life legacy archive (162.243.163.221).

=== MAJOR SOFTWARE WORKS ===

1. thePRBAT — Polygon Replicating Bitmap Authoring Tool (2011)
   Type: Desktop application (Flash/SWF-based)
   Description: Early digital imaging software created during BFA at Ringling College. A bitmap authoring tool using polygon replication as its core interaction model.
   Archive: bin-sc.jas.life/Projects/thePRBAT/

2. No Paint (2014–2020)
   Type: Web app (Rails, 2014) → iOS app (2019–2020)
   Description: Constraint-based drawing application. Originally a Ruby on Rails web app at nopaint.org (2014), rebuilt as an iOS app released in 2019–2020. The constraint: you can only paint, never erase.
   Archive: nopaint-org.jas.life (original Rails app preserved)
   Live: nopaint.art (iOS)

3. JeffreyPaint (2014)
   Type: macOS desktop application (.dmg)
   Description: A native macOS painting application with multiple builds. Created during or shortly after Linked by Air tenure.
   Archive: bin-sc.jas.life/Projects/JeffreyPaint/ (multiple .dmg builds)

4. Shrub (2014)
   Type: iOS application
   Description: iOS app created while working at Linked by Air design studio in NYC (2013–2014).

5. Finger Quilt (2016)
   Type: iOS application
   Description: Quilting-inspired touch drawing tool for iOS.

6. ShelbySoft (date unknown, pre-2016)
   Type: macOS applications
   Description: A software suite including VoiceRecorder.app and VoiceWriter.app.
   Archive: bin-sc.jas.life/Projects/ShelbySoft/

7. TDA — Teacher Directed Activity (date unknown, pre-2016)
   Type: Web application
   Description: Educational software project, related to teacherdirectedactivity.com domain.
   Archive: bin-sc.jas.life/Projects/TDA/, also /var/www/teacherdirectedactivity.com/
