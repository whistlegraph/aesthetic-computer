Installation Instructions

Method 1: Direct Install (Easiest - Safari only)

  1. Open this page in Safari on your iOS device (Chrome/Firefox won't work for installation links)
  2. Tap the blue "Install on iOS Device" button above
  3. Tap "Install" when prompted
  4. The app icon will appear on your home screen with a grayed-out appearance
  5. Go to Settings ‚Üí General ‚Üí VPN & Device Management
  6. Under "Developer App", tap on "Apple Development: make@false.work"
  7. Tap "Trust Apple Development: make@false.work"
  8. Tap "Trust" again in the confirmation dialog
  9. Return to home screen - the app icon should now be active

Method 2: Install via Mac with Xcode

  1. Download the IPA file using the link above
  2. Connect your iOS device to your Mac via USB
  3. Open Xcode ‚Üí Window ‚Üí Devices and Simulators
  4. Select your device from the left sidebar
  5. Click the "+" button under "Installed Apps"
  6. Browse to and select the downloaded SpiderLily-latest.ipa file
  7. Follow steps 5-9 from Method 1 to trust the developer profile

Method 3: Install via Mac with Finder

  1. Download the IPA file using the link above
  2. Connect your iOS device to your Mac via USB
  3. Open Finder - your device should appear in the sidebar
  4. Click on your device
  5. Drag and drop the .ipa file onto the Files section
  6. Follow steps 5-9 from Method 1 to trust the developer profile
