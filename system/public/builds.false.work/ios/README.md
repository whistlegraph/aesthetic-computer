# iOS Builds

IPA files are not committed to git due to their large size (800+ MB).

## Accessing the Build

Users can install from: **https://builds.false.work/ios/**

## Upload Instructions

The IPA file must be manually uploaded to the web server since it's too large for git.

Current build details:

- Team ID: F7G74Z35B8

## Installation Methods

Users have three options:

1. **Direct Install (Safari only)** - Open https://builds.false.work/ios/ in Safari and tap "Install on iOS Device"
2. **Xcode** - Download IPA and install via Window → Devices and Simulators
3. **Finder** - Download IPA and drag to device in Finder sidebar

After installation, users must trust the developer profile in Settings → General → VPN & Device Management.
