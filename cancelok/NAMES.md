# cancelok ‚Äî running list of project names

The app outgrew "cancelok" (there are no cancel/ok buttons anymore ‚Äî it became a
wander through a web of colour-coded pad-rooms). This is the shortlist we've been
turning over, kept so we don't re-check the same words. Domains noted where known;
availability drifts, so re-check before buying.

## Leading

- **ganny** ‚Äî from **G**raphic **A**nimated **N**otation (the pads literally are
  animated notation: the visual is the score). Reads as a mascot/creature name,
  clean, no collisions. `.app` $8.75, `.art` $3.60 free; `.com` taken. Faint
  "granny" echo, soft in the singular.
- **schizholes** ‚Äî evocative ("holes" fits the wander-through-rooms), fully coined
  and unmistakably ours. `.com`/`.app`/`.art` ALL free (rare). Flag: `schiz-` reads
  as schizophrenia ‚Äî stigma-adjacent; a deliberate edge, not an accident.
- **piece.place** ‚Äî literal, clean domain-hack. `piece.place` $18, `pieceplace.app`
  $8.75 free; `pieceplace.art` taken.

## Considered, with baggage

- **apads** ("aesthetic pads") ‚Äî reads as "iPads" (Apple adjacency + confusion).
  `.app` $8.75, `.art` $3.60 free; `.com` taken.
- **ganjam** ("GAN jam") ‚Äî reads as "ganja"; "GAN" also = generative adversarial
  network. Two wrong associations. `.app` $8.75 free; `.com` taken.
- **quilt** ‚Äî perfect metaphor (patchwork grid of pads) but a common word: `.com`/
  `.app` taken, `.art` a $348 premium. Dead end.
- **buzzer** ‚Äî `.com`/`.app`/`.io` taken, `.art` $348 premium. Dead end.
- **whistlegraph** ‚Äî great fit for "graphic animated notation," but it's ALREADY
  our own live project (whistlegraph.org). Would collide, unless this folds in.
- **slablit / slabbit** ‚Äî "slab" collides with our fleet tooling (slab menubar,
  slab/bin). slablit free; slabbit `.com`/`.app` taken.
