## Pattern for new applications

Each application folder should contain:

```
<name>/
  RESEARCH.md          # what we found about the call: deadline, ask, jury, fit
  DRAFT.md             # the canonical narrative source (form fields)
  budget.md (or .tex)  # if applicable
  attachments/         # CV, work samples, references
```

Mirror the structure of `grants/lacma-2026/` (esp. `LACMA-2026-APPLICATION-DRAFT.md`) for narrative-heavy submissions.
