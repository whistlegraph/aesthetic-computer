# Source notes — A Record in the Mail

Research checked July 18, 2026. The essay distinguishes documented API behavior from service claims and favors primary vendor documentation.

- Kunaki, *Vinyl manufacture and fulfillment*: <https://kunaki.com/product-vinyl.html>. One-unit manufacturing, direct shipping, 12-inch format, 20-minute-per-side limit, artwork and audio requirements.
- Kunaki, *HTTP Web Service*: <https://kunaki.com/httpservice.htm>. Shipping quotes, manufacture/fulfillment orders, status and tracking, prefunding, and the boundary that new titles must first be published through Kunaki's software.
- Kunaki, *XML Web Service*: <https://kunaki.com/xmlservice.htm>. Equivalent structured ordering flow and product/order identifiers.
- Kunaki, *FAQ*: <https://kunaki.com/faq.html>. Funding, pending orders, turnaround, cancellation, returns, and artwork requirements.
- Tape On Demand: <https://tape-on-demand.github.io/>. Public description of one-at-a-time cassette production, reorder/drop-shipping workflow, Bandcamp fulfillment, eight-business-day shipping claim, and indicative unit economics. No public general ordering API was found.
- Leerecs, *Physical Media On-Demand*: <https://leerecs.net/>. Early-access comparison point advertising no-minimum vinyl, cassette, and CD production; not treated as a presently open API.
- Aesthetic Computer podcast pipeline: `marketing/podcast/`. Used for the episode-to-physical-release proposal and the existing public MP3/PDF artifacts.

Prices and service availability can change. Obtain a live invoice before funding an order.
