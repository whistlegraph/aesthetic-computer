# Project Description (max 2,000 characters)

Aesthetic Computer is a mobile-first runtime and social network for creative computing that treats the programming interface as a musical instrument. Instead of an IDE, code editor, or app store, users encounter a single text prompt. They type short, memorizable commands to launch interactive programs called "pieces" — drawing tools, sound instruments, games, generative artworks, and social spaces — all running in the browser on any device.

The interaction model inverts conventional software design. There are no menus, no file browsers, no onboarding flows. Users discover commands through exploration, build muscle memory through repetition, and improvise by recombining pieces fluently — the same way a musician learns an instrument. Each URL is a playable surface: aesthetic.computer/notepat, aesthetic.computer/prompt, aesthetic.computer/wand.

The platform includes KidLisp, a minimal Lisp dialect designed for generative art that doubles as a natural interface for human-AI co-authoring. Over 16,000 KidLisp programs have been written — many by children and beginners collaborating with language models to create visual art through code they can read and understand.

Social infrastructure is built into the runtime itself: user handles, real-time chat, mood posts, and live profiles exist as part of the system, not as external services. Multi-user sessions allow collaborative drawing, music-making, and shared creative experiences in real time.

The system comprises 63,000 lines of open-source runtime code, 354 built-in pieces, 265 user-published works, and 2,800+ registered users. It runs entirely in the browser with no installation. A pack system bundles any piece into a single offline HTML file for distribution via email, USB, or blockchain.

Aesthetic Computer proposes that creative software can be organized around memorization and improvisation rather than navigation and configuration — and that this produces a fundamentally different relationship between people and computation.
