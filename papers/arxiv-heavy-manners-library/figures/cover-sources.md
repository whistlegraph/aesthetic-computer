# Cover-prompt sources — Heavy Manners Library

> The current cover is a placeholder cream-paper PNG (OpenAI billing limit hit before regen). When billing is raised, run `node papers/bin/gen-cover.mjs arxiv-heavy-manners-library --force` and the prompt below will produce the canonical cover.

## Visual elements → sources

| Element | Source |
|---|---|
| **Floating 7-inch dub-reggae record sleeve** | **Direct citation of the org's name origin**: "Heavy Manners" comes from **Prince Far I's *Under Heavy Manners*** (1976, Joe Gibbs / Errol Thompson production, Justice Album / Trojan release). The album title refers to Jamaican Prime Minister Michael Manley's 1976 State of Emergency declaration. The sleeve is the most direct visual citation of the name. Source: [Wikipedia: Under Heavy Manners (Prince Far I album)](https://en.wikipedia.org/wiki/Under_Heavy_Manners), [Discogs entry](https://www.discogs.com/master/40128-Prince-Far-I-Under-Heavy-Manners). |
| **Hand-stencilled, slightly smudged sleeve typography** | Heavy Manners Library's **printed-matter aesthetic**: silkscreen, risograph, hand-lettered zine covers. They host a Risograph Printing 101 workshop and run a Heavy Manners Zine Fair (2026 edition Apr 11–12). Source: [@heavymannerslibrary on Instagram](https://www.instagram.com/heavymannerslibrary/). |
| **Library checkout pocket + date-due slip glued to the sleeve** | The "library" half of the org's name made literal. HML is a public reading room with a circulating-style ethos for radical printed matter (zines, artist books, leftist political writing, sound recordings). The checkout pocket reads as both archive-affordance and DIY-library aesthetic. |
| **Age-yellowed corners + diagonal crease** | The org's collection orientation: not a pristine archive but a **used, handled** collection. The wear reads as "well-loved," not damaged. |
| **Faded periphery into cream paper** | Vignette; deliberately omits the surrounding 1200 N Alvarado context (Echo Park streetscape, the building exterior). |

## Founder + key people referenced

- **Matthew James-Wilson** — co-founder, 2021. [About page](https://heavymannerslibrary.com).
- **Molly Soda** (Amalia Soto) — co-founder, co-curator. Internet artist, post-internet zine maker. [Wikipedia: Molly Soda](https://en.wikipedia.org/wiki/Molly_Soda).
- Six additional staff per the about page including **Maya Man as Web Designer**. *Cross-reference SFPC dossier — Maya Man is also one of three named SFPC individual supporters.*

## Building tenancy chain (cross-reference Machine Project dossier)

The HML space is **1200 N Alvarado St, Unit D, Echo Park, Los Angeles** — the same building that housed:
- **Echo Park Film Center** (2002–January 2022, basement)
- **Machine Project** (2003–January 2018, storefront)
- **Heavy Manners Library** (2021–present, Unit D)

The Echo Park Film Center Collective continues to host screenings in HML's basement. This programmatic continuity is not inferred — it's documented on HML's own programming pages.

## Note on prompt detail

The prompt says "the centre label barely showing, its colour a faded brick-red under a layer of dust" — this is a specific reference to **Joe Gibbs' record-label color scheme** (Joe Gibbs / Errol Thompson productions used distinctive red/orange center labels). Source: [Joe Gibbs Records discography](https://www.discogs.com/label/8895-Joe-Gibbs-Records).

## What was deliberately excluded

- **Any Prince Far I portrait or likeness** (would violate the no-logos / no-faces rule and could read as appropriation).
- An explicit reggae genre signifier (Rasta colors, etc.) — too literal.
- The Echo Park neighborhood context.
- Heavy Manners' own logo (would break the no-text rule).
