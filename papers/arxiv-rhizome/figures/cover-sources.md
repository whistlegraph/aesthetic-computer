# Cover-prompt sources — Rhizome.org

The cover is a **single floating tangle of underground rhizome runners** with a small CRT-monitor glyph nested in the center. Each element traces to a real reference.

## Visual elements → sources

| Element | Source |
|---|---|
| **Tangled rhizome-root form** | The org's name. **Mark Tribe** founded the Rhizome email list in Berlin, 1996, naming it after Deleuze & Guattari's "rhizome" figure (the introduction to *A Thousand Plateaus*, 1980). The visual literalizes the figure — sinuous root tendrils crisscrossing each other in a sculptural cluster. Source: [A Thousand Plateaus](https://www.upress.umn.edu/book-division/books/a-thousand-plateaus), Plant's *Zeros and Ones* (1997), Galloway's *Protocol* (2004). |
| **Network-cable diagram echo in the runner pattern** | Galloway's [*Protocol*](https://mitpress.mit.edu/9780262072472/protocol/) (MIT Press, 2004) explicitly distinguishes the Deleuzian rhizome from TCP/IP topology. The cover threads both readings: the runners look botanical AND like a network graph, refusing to resolve. |
| **CRT monitor glyph nested in the dense center** | Net-art era visual language. Rhizome's ArtBase (launched 1999) preserves born-digital works that originally ran on CRT-era hardware. The CRT is the period-marker that says "preservation" without saying it. |
| **Floppy-disk shapes drifting near the edge** | Same period-marker logic. ArtBase's earliest acquisitions (e.g., JODI's *OSS/****/wwwwwwww.jodi.org*, 1995-) were distributed on floppy. |
| **Faded periphery dissolving into cream** | Vignette aesthetic; the rhizome is the iconic form, the periphery is uncited (intentional). |

## Founder + key people referenced

- **Mark Tribe** (b. 1966) — founder, 1996. Not visually depicted (cover prioritizes the figure over the man). [Wikipedia](https://en.wikipedia.org/wiki/Mark_Tribe).
- **Lauren Cornell** — director 2006–2012. Co-edited *Mass Effect* (MIT Press, 2015), the org's theoretical statement of the post-internet moment.
- **Heather Corcoran** — director 2012–2016.
- **Zachary Kaplan** — ED 2016–2024.
- **Michael Connor** — current ED.

## What was deliberately excluded

- The 235 Bowery building (Rhizome is co-located with the New Museum; the building belongs visually to the New Museum / NEW INC dossier, not Rhizome).
- Webrecorder UI / specific software screenshots.
- Any ArtBase-piece imagery (would imply curatorial endorsement of one work over another).
