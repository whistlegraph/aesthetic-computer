## Abstract seed

Aesthetic Computer is a creative computing environment made of small playable
pieces, public writing, and a runtime that spans browser and native hardware.
My current research asks how such an environment can support computational
creativity without becoming only a software platform or only an artwork. The
work combines KidLisp, a minimal language for generative pieces; Notepat, a
keyboard-driven visual and musical instrument; a native OS path for low-latency
performance; and a papers platter that turns ongoing experiments into papers,
cards, and dossiers. The project treats writing as part of the system: an idea
is not finished when it runs, but when it can be explained simply enough to
travel. At ICCC, I want to clarify this practice as an early-career research
program: creative tools as living instruments, with code, publication, and
community all visible in the same frame.
