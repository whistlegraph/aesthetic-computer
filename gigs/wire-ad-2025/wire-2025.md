## Visual Style
- **"Matrix smorgasbord around the edges bleeding out"**:
    - Expand the canvas to create margins.
    - Fill margins with digital noise, code snippets, and "matrix" rain effects.
    - Bleed these effects into the main image.
    - **DENSITY**: High density of text, handles, and code.
- **Typography**:
    - Use a pixel font or terminal-style font (like the one used in AC).
    - Colors: High contrast (Green/Black, Orange/Cream as seen in `laer-klokken` theme).

## Execution Steps

1.  **Generate QR Code**:
    - Create a QR code pointing to `https://aesthetic.computer/`.
    - Save as `temp_qr.png`.

2.  **Image Processing (Python Script)**:
    - Load `wire-2025.png`.
    - Expand canvas size (e.g., add 20-30% padding).
    - Generate "Matrix" background texture for the padding.
    - Overlay chat messages in speech bubbles or terminal lines.
    - Scatter user handles, musical prompts, **kidlisp code**, **moods**, and **painting codes** around the "bleeding" edges.
    - Composite the QR code into a prominent but integrated spot.
    - Apply a pixelation or color-grading filter to unify the look.

3.  **Output**:
    - Save as `system/public/assets/pruttipal/ads/wire-2025-ac.png`.
