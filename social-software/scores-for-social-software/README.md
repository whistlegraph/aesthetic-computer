## Catalog rules

- `observed` means visible/audible in the media.
- `identified` means supported by a filename, metadata, event program, email, text, or participant confirmation.
- `inferred` means plausible but awaiting confirmation.
- Descriptions should name actions, interfaces, spatial arrangements, and transitions before attempting interpretation.
- People should not be identified from appearance alone.
- Preserve original filenames and SHA-256 hashes as the stable identity of every asset.
