I'm Jeffrey Alan Scudder — artist, educator, and solo developer. I teach emerging digital practices and make open source creative tools. Since 2017 I've built painting software, live coding environments, and performance instruments — from [No Paint](https://nopaint.art) to [@whistlegraph](https://www.tiktok.com/@whistlegraph) (2.5M+ followers on TikTok) to [Aesthetic Computer](https://aesthetic.computer), a browser runtime I've been developing full-time since 2021.

Everything ships open source in one monorepo: the graphics engine, session servers, a custom Lisp dialect called KidLisp, and 350+ interactive pieces. The platform supports ~2,800 registered users — artists, students, and a dedicated community of Danish chatters — who've made over 4,300 paintings, 16,000 KidLisp programs, and 18,000 chat messages together.

My daily work is writing code, making generative art, and keeping the infrastructure alive for this community. Sponsoring directly supports that.
