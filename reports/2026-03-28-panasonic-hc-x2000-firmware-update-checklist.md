# Panasonic HC-X2000 Firmware Update Checklist (2026-03-28)

## Short version

Use Panasonic's official **SD card update** workflow.

As of **March 28, 2026**, Panasonic's HC-X2000 support page lists **Ver. 1.6** as the latest firmware, dated **February 26, 2025**.

## Checklist

1. Plug the HC-X2000 into wall power.
2. On the camera, check the installed firmware:
   - `MENU` -> `OTHERS` -> `INFORMATION` -> `VERSION`
3. On a computer, download the latest HC-X2000 firmware from Panasonic:
   - https://av.jpn.support.panasonic.com/support/global/cs/e_cam/download/x2000/index.html
4. Unzip the Panasonic download.
5. Copy the firmware update file to a clean SD card.
6. Insert that SD card into **card slot 1** on the camera.
7. On the camera, start the update:
   - `MENU` -> `OTHERS` -> `INFORMATION` -> `UPDATE`
8. Wait for the update to finish completely.
9. Check the firmware version again:
   - `MENU` -> `OTHERS` -> `INFORMATION` -> `VERSION`
10. Confirm the camera now reports **Ver. 1.6**.

## Safety rules

- Do not run the update on a low battery.
- Do not power the camera off during the update.
- Do not remove the SD card during the update.
- Use **slot 1** for the firmware card.

## Notes

- Panasonic's support material shows the update is performed by loading the firmware from an SD card in the camera.
- One Panasonic page still contains older wording about updating if the installed firmware is `1.3` or lower, but Panasonic's current HC-X2000 firmware page also lists **Ver. 1.6** as the current release. The checklist above assumes **1.6** is the target version.
